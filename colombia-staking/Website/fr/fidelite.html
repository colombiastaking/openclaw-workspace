Colombia Staking présente le token COLS : rare, porté par la communauté et avec des incitations à long terme pour les délégateurs.


30% des revenus de l'agence sont utilisés pour acheter des COLS :

—> 1/3 pour les COLS mis en staking dans notre organisation décentralisée. (Récompenses quotidiennes).

—> 2/3 pour les délégateurs de eGLD avec des COLS en staking  (Récompenses quotidiennes).









Les récompenses dans la DAO sont distribuées via le protocole Peerme, avec un verrouillage minimum de 15 jours lorsque les COLS sont mis en staking. Les COLS peuvent être retirés après 15 jours.


Les récompenses pour les délégateurs de eGLD qui ont des COLS en staking sont directement distribuées dans leur portefeuille.  Le ratio COLS/eGLD est calculé pour chaque adresse. Plus le ratio est élevé plus les intérêts distribués pour les eGLD délégués sont importants. 


Utilités du token COLS :  

1. Voter et proposer dans la DAO de Colombia Staking sur Peerme.

2. Bonus de % d'APR pour le staking de COLS.

3. Bonus de % d'APR pour déléguer ses eGLD avec des COLS en staking.

4. Token rare avec de forts rachats à partir des frais collectés par l'agence.
