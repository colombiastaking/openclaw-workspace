## Sandbox Enhanced - Blockchain Queries

### New Functions Added
- `getTaxReport(address, year)` - EGLD staking rewards tax report with weekly/monthly breakdown in USD
- `getHistoricalPrices(tokenId, days)` - Historical price data from CoinGecko
- `getPriceAtDate(tokenId, date)` - Price at specific date
- `estimateGas(operation)` - Estimate transaction gas costs
- `previewTransaction(operation, address, params)` - Build unsigned transaction
- `searchBlocks(query)` - Search blocks by hash/nonce
- `getAccountHistory(address, tokenId)` - Token transfer history
- `getTaxReportDetailed(address, year)` - Full tax report with weekly breakdown
- `getTokenHolders(tokenId)` - Token holders list
